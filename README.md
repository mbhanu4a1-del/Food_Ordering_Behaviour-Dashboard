# Food Ordering Behavior Dashboard

## 📊 Project Overview
This project analyzes customer food ordering behavior using Power BI.

## 📁 Files Included
- Power BI Dashboard (.pbix)
- Dataset (CSV)

## 📈 Key Insights
- Order trends by time of day
- Customer behavior analysis
- Revenue and performance metrics

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
